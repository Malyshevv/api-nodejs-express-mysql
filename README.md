# API методы
- /news/id - mthod: GET, info: singl post
- /news/ - mthod: GET,  info: all posts
- /news/ - mthod: POST,  info: insert post
- /news/id - mthod: DELETE,  info: delete post
- /news/id - mthod: PUT,  info: update post
